# Java 项目开发任务说明

## 项目概述

开发一个基于 Java 的学生管理系统，实现用户管理、学生信息管理、课程与成绩管理、成绩统计与分析等功能。

系统需要支持角色权限控制，不同用户有不同操作权限。

---

## 一、用户与权限管理

### 1. 用户类型

* 学生
* 教师
* 管理员

### 2. 功能需求

* **登录 / 登出**：支持用户名密码验证，登录后根据角色显示功能菜单。
* **基本信息维护**：

  * 用户信息：用户名、密码、联系方式、角色
  * 支持用户信息修改
* **角色权限控制**：

  * 学生：只能查看自己的信息和成绩
  * 教师：可管理自己任课的课程和学生成绩
  * 管理员：拥有系统全部管理权限，包括用户、学生、课程、成绩管理

---

## 二、学生信息管理

### 1. 学生基本信息

* 学号
* 姓名
* 班级
* 专业

### 2. 功能需求

* 查询学生信息
* 修改学生信息
* 删除学生信息
* **批量导入**：支持从 Excel 文件导入学生信息

---

## 三、课程与成绩管理

### 1. 课程管理

* 课程名称
* 学分
* 任课教师

### 2. 成绩管理

* 成绩录入（按学生或按课程录入）
* 成绩修改
* 成绩查询：

  * 按学生查询
  * 按课程查询
  * 按班级查询
* **批量导入导出**：支持 Excel 导入成绩数据，导出成绩报表
* **历史成绩记录**：保留每次成绩修改的历史版本，支持查看历史记录

---

## 四、成绩统计与分析（传统统计）

### 1. 统计指标

* 平均分
* 最高分 / 最低分
* 及格率
* 成绩分布（区间统计，例如 0-59、60-69、70-79、80-89、90-100）
* 成绩趋势（按时间或学期变化）

### 2. 数据可视化

* 成绩分布图（柱状图或饼图）
* 成绩趋势图（折线图）

---

## 五、系统设计要求

### 1. 技术栈

* **后端**：Spring Boot
* **数据库**：MySQL 
* **前端**：可选（若需要界面展示，可用 Thymeleaf 或 Vue/React）
* **Excel 导入导出**：使用 Apache POI 或类似库
* **图表生成**：可用 JFreeChart 或前端绘图库

### 2. 数据库设计

* **用户表**（user）：id、用户名、密码、角色、联系方式
* **学生表**（student）：id、学号、姓名、班级、专业
* **课程表**（course）：id、课程名、学分、教师id
* **成绩表**（score）：id、学生id、课程id、成绩、记录时间
* **历史成绩表**（score_history）：id、成绩id、修改前成绩、修改后成绩、修改时间

### 3. 访问权限控制

* 使用 Spring Security 或自定义拦截器，根据角色控制接口访问权限
* 确保学生只能操作自己的数据，教师只能管理自己课程的成绩，管理员全权限

### 4. 其他要求

* 所有批量导入功能需提供模板，导入失败需返回具体错误信息
* 数据修改操作需有日志或历史记录
* 前端页面和接口需简洁清晰，方便测试和演示

---
